# Rich Text Composer

Runtime rich-text document format with editor modal, `URichTextComposerDisplay`, typewriter, hyperlink/image broadcasts, and bindable **Function** chips (Blueprint-callable UFUNCTION targets).

**Engine compatibility:** **Fab / marketplace listing** and **`EngineVersion`** in **`RichTextComposer.uplugin`** (when set).
